Huntington Place HOA — Website Admin

Use this page to update meeting dates and document links. No coding required.

Step 1

Load your current data file

Click the button below and select site-data.js from your website folder. This loads your current meetings and documents into the form below.

Step 2a

Meetings & Events

Edit the dates, times, and locations for each event. Click + Add Row to add a new event.

Event Name Date Time Location
Step 2b

Document Categories & Links

Each box below is one category in the dropdown on the website. Edit document names and paste in Google Drive share links (or any link).

Tip: To get a shareable Google Drive link, right-click the file in Drive → "Share" → "Copy link". Make sure the access is set to "Anyone with the link."
Step 3

Save & Download

When you're done editing, click the button below. A file called site-data.js will download automatically.

Then replace the old site-data.js in your website folder with this downloaded file. The website will show your changes immediately.
